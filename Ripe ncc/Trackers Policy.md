Cookies used by the RIPE NCC
============================

This page may also cover other non-cookie technologies (e.g. anonymous device fingerprinting, anonymous unique identifiers).

For more information on how we use cookies, check our [Privacy Statement](https://www.ripe.net/about-us/legal/ripe-ncc-privacy-statement#cookies).

Strictly necessary cookies
--------------------------

| Cookie name | Purpose | Term |
| --- | --- | --- |
| **crowd.token\_key** | This cookie identifies the user. It is used to authenticate logged-in single-sign-on (RIPE NCC Access) users. | Expires when the browser session is closed |
| **crowd.ripe.hint** | This cookie is used to identify logged-in users on RIPE NCC services (Single Sign On). | Expires when the browser session is closed |
| **PHPSESSID** | This is a session cookie, used to identify a user's session on a website. It is commonly used in PHP-based web applications to store a unique identifier for a user's session on the server. | Expires when the browser session is closed |
| **JSESSIONID** | This is a general-purpose platform session cookie, used by sites written in JSP. Usually used to maintain an anonymous user session by the server. | Expires when the browser session is closed |
| **\*\_sessionid** | This cookie is used to authenticate a user's logged-in status and permissions. | Expires after 2 weeks |
| **\*\_csrftoken** | This cookie is associated with the Django web development platform for Python. It is designed to help protect a site against a particular type of software attack on web forms. | Expires after 1 year |
| **MoodleSession** | You must allow this cookie used by our learning platform Moodle into your browser to provide continuity and maintain your login from page to page. | Expires when the browser session is closed |
| **\_\_cf\_bm** | The \_\_cf\_bm cookie is a cookie necessary to support Cloudflare Bot Management, currently in private beta. As part of our bot management service, this cookie helps manage incoming traffic that matches criteria associated with bots. | Expires after 30 minutes |
| **cf\_chl\_cc\_xxxxxxxxxxxx** | These cookies are used by Cloudflare for the execution of Javascript or Captcha challenges. They are not used for tracking or beyond the scope of the challenge. | Expires when the browser session is closed |
| **cf\_chl\_2** | These cookies are used by Cloudflare for the execution of Javascript or Captcha challenges. They are not used for tracking or beyond the scope of the challenge. | Expires after a few seconds |
