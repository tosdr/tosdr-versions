* **What is Uploadrar?**  
      
    Uploadrar is a file hosting provider. We offer online storage/remote backup capacity, sophisticated uploading and downloading tools.  
    With Uploadrar you can host files, images, videos, audio and flash on the same place.  
      
    
* **Why should I use Uploadrar?**  
      
    Whenever you need to send a file that is too large for e-mail, Uploadrar can help. If you need secure remote storage capacity for off-site backups, Uploadrar offers solutions for you. If you want to access personal data from a variety of computers and don't want to carry around a USB stick, Uploadrar is a perfect way of doing so.
